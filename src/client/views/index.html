How to read your result:

Sentiment indicates the polarity found (or not found) in the element it refers to
Agreement the different elements have the same polarity OR DISAGREEMENT: there is disagreement between the different elements' polarity.
Subjectivity marks the subjectivity of the text. It has two possible values: OBJECTIVE: the text does not have any subjectivity marks. SUBJECTIVE: the text has subjective marks.
Irony indicates the irony of the text. NONIRONIC: the text does not have ironic marks IRONIC: the text has ironic marks
